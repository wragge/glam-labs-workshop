# GLAM Labs Workshop

Files used in the 'Hacking the Archive' workshop held in Edinburgh on 24 June 2026. See the [workshop slides](https://slides.com/wragge/glam-labs-workshop) for more information.

## Bookmarklets

- [`bookmarklet.js`](https://github.com/wragge/glam-labs-workshop/blob/main/bookmarklet.js) – a simple bookmarklet that searches for digitised images from the National Library of Scotland collection in Wikimedia Commons
- [`bookmarklet-minified.js`](https://github.com/wragge/glam-labs-workshop/blob/main/bookmarklet-minified.js) – a minified version of the above bookmarklet

## Userscripts

- [`my-first-script.js`](https://github.com/wragge/glam-labs-workshop/blob/main/my-first-script.js) – a very basic userscript that changes the text on a button on the National Library of Scotland homepage
- [`nls-wiki-links.user.js`](https://github.com/wragge/glam-labs-workshop/blob/main/nls-wiki-links.user.js) – a userscript that retrieves and displays information about how digitised images from the National Library of Scotland collection are used and linked on Wikimedia sites
- [`nls-home-thumbnails.user.js`](https://github.com/wragge/glam-labs-workshop/blob/main/nls-home-thumbnails.user.js) – a userscript that adds a random-ish selection of thumbnail images from the National Library of Scotland collection to the home page (see [the slides](https://slides.com/wragge/glam-labs-workshop) for configuration options)
- [`nls-add-catalogue-info.user.js`](https://github.com/wragge/glam-labs-workshop/blob/main/nls-add-catalogue-info.user.js) – userscript to add an extra row to the display of catalogue information in the National Library of Scotland's Primo interface
